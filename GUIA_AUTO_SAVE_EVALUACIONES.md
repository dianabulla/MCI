# Guía: Auto-Save en Evaluaciones

## ¿Qué cambia?

Anteriormente, tenías que hacer clic en **"Guardar evaluación"** para guardar tu evaluación después de crear las preguntas.

Ahora: **Las evaluaciones se guardan automáticamente** conforme escribes. No necesitas hacer clic en guardar.

## Cómo Funciona

1. **Mientras escribes el título**: Espera 3 segundos después de dejar de escribir
2. **Al cambiar nivel, módulo, lección**: Se guarda inmediatamente
3. **Al agregar una pregunta**: Se guarda en ~1.5 segundos
4. **Al cambiar respuesta correcta**: Se guarda en ~2 segundos
5. **Al eliminar una pregunta**: Se guarda automáticamente

## Indicador de Estado

A la derecha del título "Crear evaluación" verás un indicador:

- **Sin mensaje**: No hay cambios pendientes
- **⏳ Guardando...**: Se está enviando la evaluación al servidor
- **✓ Guardado automático**: Evaluación guardada exitosamente
- **✗ Error al guardar**: Algo salió mal (revisa la conexión)

## Que NO Necesitas Hacer

- ❌ No necesitas esperar a que termine de escribir antes de cambiar de campo
- ❌ No necesitas hacer clic en "Guardar evaluación"
- ❌ No necesitas preocuparte por perder cambios

## Que SÍ Funciona

✓ Crear título y descripción
✓ Cambiar nivel, módulo, lección
✓ Agregar/eliminar preguntas
✓ Cambiar opciones (A, B, C, D)
✓ Cambiar respuesta correcta
✓ Configurar fechas de habilitación (si tienes permiso)

## Ventajas

- **Más seguro**: No perderás cambios si se cierra accidentalmente el navegador
- **Más rápido**: No necesitas esperar pantalla de confirmación
- **Mejor experiencia**: Sin interrupciones, solo trabajas

## Ejemplo de Flujo

1. Escribes título "Mi Evaluación"
   → Esperas 3 segundos → ✓ Guardado automático
   
2. Cambias nivel a "Nivel 2"
   → Instantáneamente → ✓ Guardado automático
   
3. Haces clic en "Agregar pregunta"
   → Escribes la pregunta y opciones
   → Esperas 3 segundos → ✓ Guardado automático
   
4. Cambias la respuesta correcta a "B"
   → Esperas 2 segundos → ✓ Guardado automático

## Solución de Problemas

### Veo "✗ Error al guardar"
- Verifica que tengas conexión a internet
- Comprueba que el servidor está funcionando
- Recarga la página e intenta nuevamente

### El indicador no aparece
- Verifica que estés editando en la zona "Crear evaluación"
- Cierra y abre nuevamente el navegador
- Intenta cambiar un campo para activar el auto-save

### No está guardando
- Verifica que escribas en los campos correctos (título, opciones, etc.)
- El auto-save requiere cambios reales (escribir texto)
- No guarda si los datos son iguales a los anteriores

## Nota de Seguridad

El auto-save respeta todos tus permisos:
- Solo administradores pueden crear evaluaciones
- Los datos se validan como siempre
- La sincronización es automática y segura
